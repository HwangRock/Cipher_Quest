# 프로젝트 관련
GDGoC 3기 정보보안 스터디에서 진행한 암호 퀴즈 프로젝트.  

한 사람이 스테이지를 정하고 평문을 작성해서 퀴즈를 내고 한 사람이 암호문을 복호화하면서 퀴즈가 진행됩니다.  

암호 알고리즘을 서비스로 어떻게 활용할 수 있을까 고민하면서 기획과 서비스 개발을 시작했습니다.
